# Loan-Status-Prediction

This project predicts loan status using machine learning techniques.
Its predicts if a person is eligible for a loan or not based on various features.
The dataset used contains information about applicants for example if the person is Graduate, married,..., and their loan status.

# Stack

  - Python

# Libraries

  - Numpy
  - Pandas
  - Seaborn
  - Matplotlib
  - Sklearn

# Machine Learning Model

  - Support Vector Machine Model (SVM)
